# 05 Income — Redesign

This is the headline "what AI got wrong, how I caught it" story for Module 5 Path A.

The expected arc:

1. **First attempt (probably wrong):** ask AI to generate income variants the obvious way (suits / casual / worn clothing).
2. **Audit findings:** document why that approach fails — Western-class signifiers don't translate globally; attire-as-class risks caricature; "worn clothing = poor" reinforces stereotype.
3. **Redesign:** abstract symbol approach (coin counts, dwelling size, bar height, or another quantitative encoding). Income reads as a number, not a costume.
4. **Second attempt:** prompts for the redesigned approach. Document the iteration.
5. **Final design:** what shipped, why.

This file is the lesson. Capture honestly even if (especially if) the first attempt is embarrassing.
